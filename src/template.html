Monte Carlo Path Tracer

A Monte Carlo Path Tracer with global illumination effect using Monte Carlo estimation with multiple importance sampling. It supports various surface materials (diffuse, specular, transmissive, microfacet etc.), light sources (point, spot, area), and geometries.

Mini-Minecraft

A simple version of sandbox game, Minecraft. It includes procedurally generated terrain, rivers, and various biomes: mountains, fields, canyons etc. using Fractal Brownian Motion / Perlin noise functions and L-System.
